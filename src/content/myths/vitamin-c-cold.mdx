## 迷思的起源

維他命 C 能預防感冒的說法，最早可追溯至 1970 年代。兩屆諾貝爾獎得主 Linus Pauling 出版了《維他命 C 與普通感冒》一書，主張每日攝取 1-2 克維他命 C 可以預防甚至治癒感冒。由於 Pauling 的崇高學術地位，這一觀點迅速普及。

## 科學怎麼說？

### 對一般人的預防效果

目前最全面的證據來自 Cochrane 系統性回顧。該回顧分析了 29 項隨機對照試驗，涵蓋超過 11,000 名受試者。結論很明確：**對於一般人而言，定期補充維他命 C 無法有效降低感冒的發生率**。

風險比為 0.97（95% 信賴區間 0.94-1.00），意味著統計上幾乎沒有差異。

### 對高壓環境者的效果

有趣的是，在 5 項針對高強度運動者（馬拉松跑者、滑雪者、極地訓練士兵）的試驗中，定期補充維他命 C 確實降低了感冒發生率約 50%。這可能與劇烈運動造成的氧化壓力與免疫抑制有關。

### 對病程的影響

定期補充者一旦感冒，病程可略微縮短：

- **成人**：平均縮短約 8%（相當於少半天）
- **兒童**：平均縮短約 14%（相當於少一天）

值得注意的是，這是「預防性」定期補充的效果。感冒發作後才開始補充，效果則不一致。

## 為什麼仍然流傳？

1. **權威效應**：Pauling 的諾貝爾獎光環讓人們傾向相信其觀點
2. **確認偏誤**：人們記住感冒時吃維他命 C 後好轉的經驗，忘記不吃也會好的事實
3. **商業行銷**：保健品產業持續強調維他命 C 的免疫功能
4. **部分真實**：維他命 C 確實參與免疫功能，只是「參與」不等於「補充就能預防」

## 實際建議

- 透過均衡飲食攝取維他命 C（芭樂、奇異果、甜椒等）通常已足夠
- 無需為了「預防感冒」而額外購買維他命 C 補充品
- 若為高強度運動者，可考慮每日補充 200 毫克以上
- 感冒時補充可能略有幫助，但期望值不宜過高
- 真正有效的感冒預防措施：勤洗手、充足睡眠、規律運動
