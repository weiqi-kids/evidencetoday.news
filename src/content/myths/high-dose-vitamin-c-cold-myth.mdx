## 迷思說法

感冒時大量補充維生素 C 可以明顯縮短病程或治療感冒。

## 判斷結論

Cochrane 回顧顯示，長期規律補充維生素 C 只能小幅縮短感冒時間；出現症狀後才大量服用，通常不會明顯縮短病程。高劑量可能造成腹瀉、胃不適或腎結石風險。

## 先看懂這個詞

這篇要先分清「營養素支持免疫」與「治療感冒」。前者是維持功能，後者要看臨床效果。

## 坊間怎麼流傳，為什麼容易相信？

坊間常見的說法是：「一感冒就趕快吃大量維他命 C，可以把感冒壓下去」或「維他命 C 吃越多免疫力越強」。

這個說法容易被相信，是因為維生素 C 確實和免疫細胞功能有關。問題在於，參與免疫不等於高劑量能治療病毒感染。

發泡錠和粉包很有儀式感，症狀自然好轉時也容易被歸功於補充品。

## 哪些人要特別小心？

- **腎結石病史或腎功能不佳者：** 高劑量維生素 C 可能增加草酸負擔或結石疑慮。
- **容易胃痛、腹瀉者：** 大量維生素 C 可能加重腸胃不適。
- **發燒持續、呼吸喘或症狀明顯惡化者：** 不應只靠維生素 C 觀察，需評估是否為流感、COVID-19 或其他感染。
