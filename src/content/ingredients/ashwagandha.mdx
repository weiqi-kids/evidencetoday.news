## 30 秒認識

- 南非醉茄（Ashwagandha，學名 Withania somnifera）是印度阿育吠陀傳統醫學使用數千年的根部藥草，現代研究將它歸類為「適應原」（adaptogen），意指可能協助身體因應壓力 <a href="#ref-7">[7]</a>。
- 多篇小型隨機對照試驗顯示，連續補充含睡茄內酯的根部萃取物約8至12週，可讓受試者的壓力量表（PSS）分數較安慰劑組下降 <a href="#ref-1">[1]</a><a href="#ref-2">[2]</a>。
- 一篇整合5項RCT、共約400名參與者的統合分析發現，南非醉茄對整體睡眠品質有小幅但具統計意義的改善，確診失眠者效果較明顯 <a href="#ref-4">[4]</a>。
- 貝式統合分析顯示南非醉茄可能提升心肺適能表現，對肌力與爆發力的效果量中等且方向一致，但僅涵蓋5篇試驗、198名受試者，證據基礎仍有待擴充 <a href="#ref-5">[5]</a>。
- 美國NIH旗下資料庫與NCCIH皆提醒，南非醉茄有肝損傷通報案例、可能影響甲狀腺功能與部分藥物交互作用，孕期與哺乳期應避免使用 <a href="#ref-6">[6]</a><a href="#ref-7">[7]</a>。

## 基本介紹

南非醉茄屬於茄科（Solanaceae）植物，原生於印度、中東與地中海周邊乾燥地區，並非以南非為主要產地，中文慣用譯名「南非醉茄」與植物實際原生地並不完全對應。傳統阿育吠陀醫學使用其根部粉末，搭配蜂蜜、牛奶或酥油服用，用於增強體力、協助恢復元氣。現代保健食品則多以標準化萃取物形式販售，並以「睡茄內酯」（withanolides，一類類固醇內酯化合物）作為主要指標成分含量標示依據。

## 這是什麼？

南非醉茄的英文俗名 ashwagandha 源自梵文，字面意為「馬的氣味」，傳統上被認為指根部特有氣味，也隱含服用後可能帶來如馬一般體力與活力的說法，這類命名本身屬於傳統文化描述，不構成現代科學證據。目前市面常見的標準化萃取物品牌包括 KSM-66（僅取根部、睡茄內酯濃度約5%）與 Sensoril（根部與葉片皆取用，內酯苷含量較高），兩者製程與成分比例不同，人體研究結果不能直接互相套用到「南非醉茄」這個統稱之下的所有產品。

## 在身體裡做什麼？／主要作用機制

南非醉茄的作用機制多數證據來自動物與體外實驗，人體確認程度有限。主要假說包括：調節下視丘-腦垂體-腎上腺軸（HPA軸），可能是部分人體試驗觀察到血清皮質醇下降的原因之一 <a href="#ref-1">[1]</a>；動物與體外研究也提出其成分可能具有類似GABA受體活化的作用，理論上有助放鬆與睡眠，但這部分尚缺乏對應的人體機轉驗證 <a href="#ref-4">[4]</a>；另有小型研究與個案報告提示南非醉茄可能影響甲狀腺荷爾蒙濃度，甲狀腺疾病或正在服用甲狀腺相關藥物者應謹慎評估後再使用 <a href="#ref-7">[7]</a>。

## 食物來源／常見來源

南非醉茄並非一般飲食中常見的食物來源，日常三餐幾乎不會攝取到具意義劑量。傳統阿育吠陀配方會將根部乾燥磨粉，加入牛奶、酥油或蜂蜜調成藥膳服用，但這類傳統作法的劑量與現代標準化萃取物並不相同，難以類比換算。

## 建議攝取量／常見攝取建議／研究中常見使用方式

多數人體研究使用的劑量落在每日300至600毫克標準化根部萃取物，分1至2次服用，療程多為8至12週 <a href="#ref-1">[1]</a><a href="#ref-2">[2]</a><a href="#ref-3">[3]</a>。睡眠相關統合分析指出，每日劑量達600毫克以上、療程8週以上時，效果較為明顯 <a href="#ref-4">[4]</a>。由於南非醉茄並非人體必需營養素，目前並無官方訂定的每日建議攝取量，市售產品標示劑量主要參照上述臨床試驗設計而非法規規範。

## 缺乏風險或攝取不足問題

南非醉茄屬於植物萃取的保健成分，並非維生素、礦物質等必需營養素，人體不會因為「攝取不足」而出現特定缺乏症狀。坊間若出現「長期不補充南非醉茄會導致慢性疲勞或壓力失衡」之類的說法，並無科學根據，應理解為行銷用語而非醫學事實。

## 補充品常見形式

市售南非醉茄產品常見形式包括膠囊、錠劑、粉末沖泡包，以及與其他適應原草本（如紅景天、聖羅勒）複方的機能飲品或軟糖。同一品牌名稱下，仍可能因批次萃取比例調整而使睡茄內酯含量略有差異，選購時建議留意產品是否標示萃取比例與檢驗來源，而非只看「南非醉茄」四字。

## 安全性與注意事項

近年陸續有南非醉茄相關肝損傷的病例報告，NIH旗下LiverTox資料庫收錄多起案例，多數表現為膽汁鬱積型肝炎，症狀包含黃疸、皮膚搔癢與茶色尿，發生時間從服用後數週到數月不等；澳洲藥物管理局（TGA）也於2024年就相關肝損傷通報發出安全警訊 <a href="#ref-6">[6]</a>。美國NCCIH整理的安全性資訊指出，南非醉茄可能與鎮靜劑、抗癲癇藥物、免疫抑制劑、甲狀腺相關藥物、糖尿病與降血壓藥物產生交互作用，建議近期預定手術者避免使用；動物實驗顯示可能有胚胎毒性與子宮刺激作用，因此NCCIH等機構建議懷孕與哺乳期婦女避免服用 <a href="#ref-7">[7]</a>。茄科植物過敏者、自體免疫疾病患者（因可能刺激免疫系統）也建議先諮詢醫療人員再評估是否使用。

## 常見迷思

- **「南非醉茄可以取代抗焦慮藥物」**：現有RCT多針對健康成人的一般生活壓力與輕度焦慮情緒，並非臨床診斷的焦慮症治療試驗，兩者不能直接劃上等號 <a href="#ref-1">[1]</a><a href="#ref-2">[2]</a>。
- **「所有南非醉茄產品效果都一樣」**：KSM-66、Sensoril等萃取物在取材部位、萃取比例與睡茄內酯含量上差異明顯，個別研究結果不能套用到所有市售產品。
- **「純天然保健品不會有肝毒性風險」**：LiverTox資料庫與多起病例報告顯示，南非醉茄確實有引發肝損傷的紀錄，天然來源不等於零風險 <a href="#ref-6">[6]</a>。
- **「懷孕也可以吃來舒緩孕期壓力」**：NCCIH明確建議懷孕與哺乳期避免使用，人體安全性資料本就不足 <a href="#ref-7">[7]</a>。
- **「劑量越高效果越好」**：多數臨床試驗僅測試每日300至600毫克，更高劑量的效果與安全性缺乏對應研究支持，不宜自行加量。
