## 30 秒認識

- 蝦紅素（Astaxanthin）是一種紅橘色的類胡蘿蔔素（葉黃素家族的含氧型），讓鮭魚、蝦蟹、紅鶴呈現紅色，是脂溶性的強抗氧化劑。 <a href="#ref-1">[1]</a>
- 與 β-胡蘿蔔素不同，它**不會轉換成維生素 A**，因此沒有「過量變成維生素 A 中毒」的問題，但也不能拿來補充維生素 A。 <a href="#ref-1">[1]</a>
- 人體效益以「皮膚彈性」的證據相對較多：統合分析顯示口服可小幅改善，但作者明言**證據仍不足以作為治療光老化的建議**。 <a href="#ref-3">[3]</a>
- 天然蝦紅素安全性良好（研究支持每日達 24 毫克的天然來源），但「抗氧化力是維生素 C 數千倍」這類宣稱多是試管數據，缺乏臨床意義。 <a href="#ref-1">[1]</a>
- 「護眼」的證據以 40 歲以上的**螢幕使用族群**較明確：一項隨機對照試驗顯示每日 9 毫克、連續 6 週能改善這個年齡層使用螢幕後的矯正視力，但 40 歲以下未見顯著差異，仍屬單一小型試驗。 <a href="#ref-6">[6]</a>

## 基本介紹

蝦紅素是含氧的類胡蘿蔔素（屬於葉黃素類，xanthophyll），主要由微藻「雨生紅球藻（Haematococcus pluvialis）」合成；鮭魚、蝦蟹等海洋動物則是透過攝食藻類或浮游生物而累積它，呈現紅橘色。 <a href="#ref-2">[2]</a> 保健品的蝦紅素多來自紅球藻萃取（天然）或化學合成；兩者在安全資料上有差異，天然來源的人體安全研究較完整。 <a href="#ref-1">[1]</a>

## 這是什麼？

標示常見 Astaxanthin，中文也作蝦青素。要先建立兩個觀念：第一，它是「抗氧化劑」而非必需營養素，沒有每日必需建議量；第二，類胡蘿蔔素是一個大家族，蝦紅素、葉黃素、玉米黃素、β-胡蘿蔔素各有不同分佈與作用，不能因為都是「抗氧化」就互相代換。蝦紅素的特點是脂溶性高、能橫跨細胞膜的內外兩側。 <a href="#ref-1">[1]</a>

## 在身體裡做什麼？／主要作用機制

蝦紅素被研究的核心是抗氧化與抗發炎：它能有效淬熄單線態氧、清除自由基，且因分子結構可同時保護細胞膜的脂質層內外。 <a href="#ref-1">[1]</a> 這些是生化與機制層面的觀察，**不等於臨床療效**。人體研究多以氧化壓力指標、發炎指標、皮膚彈性或眼睛疲勞為終點，規模偏小且常由產品廠商贊助；要把「抗氧化指標改善」推論成「抗老、護眼、顧心血管」，目前證據還不足。

## 食物來源／常見來源

蝦紅素天然存在於紅色的海鮮中，含量最高的是**野生鮭魚（尤其紅鮭 sockeye）**，其魚肉每公斤約含 26–38 毫克；其次為鱒魚、蝦、蟹、磷蝦與鮭魚卵，含量較低。 <a href="#ref-1">[1]</a> 養殖鮭魚的紅色則多來自飼料中添加的蝦紅素（天然或合成）。要注意：研究使用的劑量（每日約 4–12 毫克）若要單靠飲食達成，需要相當份量的鮭魚，現實上不易，因此它在實務上仍偏向補充品議題。規律吃魚的價值更多來自整體營養（如 Omega-3、蛋白質），而非單一蝦紅素。

## 建議攝取量／常見攝取建議／研究中常見使用方式

蝦紅素沒有每日必需攝取量。臨床試驗常見每日 4–12 毫克，皮膚相關研究多在 6–12 毫克、持續 8–16 週。 <a href="#ref-3">[3]</a> 歐洲食品安全局（EFSA）2020 年訂定可接受每日攝取量（ADI）為每公斤體重 0.2 毫克，以 70 公斤成人換算約為每日 14 毫克。 <a href="#ref-2">[2]</a> 由於是脂溶性，隨含脂肪的餐食一起吃吸收較好。再次提醒：劑量越高不等於效果越好，多數正面結果也只是指標層級的小幅變化。

## 缺乏風險或攝取不足問題

蝦紅素不是必需營養素，**不存在「缺乏症」**。不吃它並不會造成任何已知的健康缺損。因此它的定位應該是「可選的抗氧化補充」，而不是「缺了就要補」。把皮膚老化、眼睛疲勞或疲倦直接歸因於「缺蝦紅素」並不合理——這些問題更受防曬、用眼習慣、睡眠與整體飲食影響。

## 補充品常見形式

市售多為紅球藻萃取的軟膠囊（油劑），常與葉黃素、玉米黃素、魚油、維生素 E 等做成護眼或抗老複方。 <a href="#ref-1">[1]</a> 複方雖看似完整，但會讓判讀變難：使用後若有變化，難以歸因；若有不適，也較難找出來源。選購時可看每份的蝦紅素毫克數、是天然（紅球藻）或合成、是否隨餐建議，以及是否有第三方檢驗，而不是被「超強抗氧化」文案主導。

## 安全性與注意事項

天然蝦紅素的安全性資料相對完整：一篇回顧 87 項人體研究（含 35 項使用每日 ≥12 毫克）未發現安全疑慮，並認為天然來源每日達 24 毫克的劑量有安全證據支持（合成形式則資料較少）。 <a href="#ref-1">[1]</a> 高劑量可能讓膚色略偏紅、出現輕微腸胃不適。其他注意：理論上可能輕微影響血壓、血糖，正在服用降血壓或降血糖藥者宜留意；與其他類胡蘿蔔素併用時注意總量；孕婦、哺乳者資料有限，不建議自行高劑量使用。

## 常見迷思

- 「蝦紅素抗氧化力是維生素 C 的數千倍」：這類數字多來自試管抗氧化測定，無法等同人體臨床效益。 <a href="#ref-1">[1]</a>
- 「吃了就能抗老、護膚、不怕曬」：皮膚彈性僅有小幅、不一致的證據，作者也認為不足以作為治療建議；它**不能取代防曬**。 <a href="#ref-3">[3]</a>
- 「天然所以可以無限量吃」：天然來源安全性雖佳，仍有 ADI 與建議劑量；過量無額外好處還可能造成不適。 <a href="#ref-2">[2]</a>
- 「能補維生素 A」：蝦紅素不會轉換為維生素 A，兩者不可互相替代。
