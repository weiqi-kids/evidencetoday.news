## 30 秒認識

- 水飛薊素（Silymarin）是從奶薊（水飛薊）種子萃取的類黃酮複合物，主要活性成分為 silibinin（水飛薊賓）。 <a href="#ref-1">[1]</a>
- 它是最熱門的「護肝」成分，但證據與行銷有落差：Cochrane 回顧**無法證實它能降低酒精性或病毒性肝病的死亡率或併發症**。 <a href="#ref-2">[2]</a>
- 對非酒精性脂肪肝（NAFLD/MASLD），統合分析顯示可小幅降低 ALT、AST，但研究品質參差、臨床意義有限。 <a href="#ref-3">[3]</a>
- 真正療效較明確的是「靜脈注射 silibinin 用於死帽蕈中毒」——那是醫院藥物，與你在藥局買的口服護肝錠不是同一件事。 <a href="#ref-1">[1]</a>

## 基本介紹

奶薊是菊科植物，其種子萃取物 silymarin 由多種類黃酮木脂素組成（silibinin、silydianin、silychristin 等），傳統上被用於肝膽保健。 <a href="#ref-1">[1]</a> 現代研究關注它的抗氧化、抗發炎與穩定肝細胞膜等可能作用，但要先建立兩個前提：第一，口服 silymarin 的生體可用率偏低；第二，「機制看似護肝」不等於「臨床上能改善肝病預後」。

## 這是什麼？

標示上常見 Milk Thistle、Silymarin 或 Silibinin，中文可能寫成奶薊、水飛薊、乳薊。 <a href="#ref-1">[1]</a> 要分清三件事：奶薊是植物、silymarin 是其萃取的活性複合物、silibinin 是 silymarin 中主要的單體。市售保健品多為標準化的 silymarin 口服製劑；醫院用於毒蕈中毒的則是純化 silibinin 的靜脈注射劑，兩者劑型、劑量與用途差距極大。

## 在身體裡做什麼？／主要作用機制

研究中討論的機制包括：清除自由基的抗氧化作用、抑制發炎訊號、穩定肝細胞膜以減少毒素進入，以及可能促進肝細胞再生。 <a href="#ref-1">[1]</a> 在毒蕈中毒，silibinin 被認為可競爭性阻斷肝細胞攝取 amatoxin，這是它作為輔助解毒的理論基礎。不過這些機制多來自實驗室與動物，**口服保健品能否在人體達到有意義的肝臟保護，仍是證據最薄弱的環節**。

## 食物來源／常見來源

水飛薊素不是一般飲食營養素，**沒有日常食物來源**——它來自奶薊植物的種子萃取，需以保健品或藥品形式攝取。 <a href="#ref-1">[1]</a> 因此和維生素不同，討論它時不存在「先從食物補足」的選項。這也提醒讀者：護肝的第一線是節制飲酒、控制體重與血糖、避免亂服來路不明藥物與保健品（這些反倒是常見的藥物性肝損傷來源），而非某種萃取物。

## 建議攝取量／常見攝取建議／研究中常見使用方式

水飛薊素沒有每日必需建議量。臨床研究與市售產品常見以標準化 silymarin 計，每日約 140 毫克、一日 2–3 次（總量約 280–420 毫克），療程數週到數月。 <a href="#ref-3">[3]</a> 由於口服吸收差，部分產品會以磷脂複合（如 silibinin-phosphatidylcholine）提升吸收。要強調的是：即便在這些劑量下，對肝病硬性結果（死亡、併發症）的效益仍未獲高品質證據支持。 <a href="#ref-2">[2]</a>

## 缺乏風險或攝取不足問題

水飛薊素不是必需營養素，**不存在「缺乏症」**。把疲倦、油膩飲食或飲酒後的不適歸因於「需要護肝」而購買補充品，是常見的行銷邏輯陷阱。真正需要評估肝臟的情況（如肝指數異常、B/C 肝帶原、長期飲酒、脂肪肝），應以抽血與醫療追蹤判斷，而不是靠保健品自我處理。

## 補充品常見形式

常見標準化 silymarin 膠囊/錠劑、磷脂複合型，以及與五味子、薑黃、朝鮮薊、B 群等做成的「護肝複方」。 <a href="#ref-1">[1]</a> 複方會讓判讀更困難，也增加交互作用的可能。選購時應看標準化的 silymarin 含量、是否有第三方檢驗，並對「解酒護肝」「修復肝臟」等宣稱保持保留——這類用語在台灣對非藥品而言也涉及誇大療效的法規問題。

## 安全性與注意事項

整體而言，水飛薊素長期耐受性大致良好，全球大量使用下僅有罕見且多為複雜情境的肝臟不良事件報告。 <a href="#ref-1">[1]</a> 常見副作用為輕微腸胃不適或輕瀉。需留意：

- **過敏**：對菊科植物（豚草、菊花、萬壽菊等）過敏者可能交叉過敏。
- **藥物交互作用**：理論上可能影響部分經肝臟代謝（CYP 酵素）的藥物，正在用藥者宜諮詢醫師或藥師。
- **血糖**：可能輕微影響血糖，糖尿病患者用藥期間應監測。
- **孕婦、哺乳者**：資料不足，不建議自行使用。

## 常見迷思

- 「護肝、排毒、解酒」：肝臟本身就是代謝器官，不需要保健品「排毒」；silymarin 也無證據能讓人「喝酒不傷肝」，這種想法反而危險。 <a href="#ref-1">[1]</a>
- 「醫學證實能治肝病」：對酒精性/病毒性肝病的死亡率與併發症，Cochrane 回顧未能證實效益。 <a href="#ref-2">[2]</a>
- 「口服護肝錠跟急救解毒一樣強」：毒蕈中毒用的是靜脈注射 silibinin（醫院藥物），與口服保健品的劑型、劑量、用途完全不同。 <a href="#ref-1">[1]</a>
- 「天然萃取所以對肝一定好」：天然不代表沒有過敏、交互作用或品質問題；肝指數異常仍應就醫，而非自行補充。
