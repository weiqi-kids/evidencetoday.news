## 30 秒認識

- 番紅花萃取（Saffron）是番紅花柱頭的濃縮萃取，市售保健品多為專利標準化原料 affron，標定活性成分番紅花素（crocins）與番紅花醛（safranal）。 <a href="#ref-1">[1]</a>
- 主打方向是助眠與情緒。助眠的主要人體證據來自 Lopresti 團隊 2020 年、63 名自述睡眠困擾成人的隨機對照試驗：affron 每日 28 毫克、連續 28 天，失眠嚴重度指數（ISI）優於安慰劑；該試驗由原料相關團隊執行。 <a href="#ref-1">[1]</a>
- 一篇彙整 8 篇隨機對照試驗的統合分析支持番紅花改善主觀睡眠品質（PSQI 平均差約 −2.14），並呈劑量反應趨勢；但納入試驗規模小、族群混雜（含健康成人、失眠者、第二型糖尿病與美沙冬維持治療者），對一般失眠者的外推有限。 <a href="#ref-2">[2]</a>
- 情緒面有明確劑量門檻：Kell 團隊 2017 年、128 名低落情緒（非臨床憂鬱）健康成人的試驗中，每日 28 毫克有效、22 毫克無效；這樣的訊號不宜外推成憂鬱症治療。 <a href="#ref-4">[4]</a>
- 安全劑量與孕婦禁忌要分清楚：保健常用約 28–30 毫克/日耐受良好，但毒理學回顧指出高劑量具子宮刺激與催經作用，孕婦與哺乳者屬明確禁忌族群。 <a href="#ref-7">[7]</a>

## 基本介紹

番紅花（Crocus sativus L.）柱頭是全球單價最高的香料之一。保健品用的番紅花萃取多為標準化原料，把柱頭中的水溶性色素番紅花素（crocins）與揮發性成分番紅花醛（safranal）濃縮並標定含量，最常見的專利原料是 affron。 <a href="#ref-1">[1]</a> 台灣市場近年把它當成助眠與情緒的差異化主打，衛教與品牌內容常見的建議劑量落在每日 14–30 毫克、早晚分兩次。 <a href="#ref-9">[9]</a>

## 這是什麼？

標示上常見 Saffron 或 Crocus sativus。先建立幾個觀念。第一，它是植物萃取的機能性成分，沒有每日必需攝取量，缺它不會生病。第二，宣稱的活性來自番紅花素與番紅花醛，保健研究幾乎都用標準化萃取（多為 affron），一般料理用的番紅花香料在劑量與純度上和研究品無法直接對應。第三，助眠與情緒是兩條不同的證據線，各自的樣本、劑量與族群都要分開看，別把其中一條的結果套到另一條上。 <a href="#ref-2">[2]</a>

## 在身體裡做什麼？／主要作用機制

番紅花被提出的機制主要有三條線。其一是調節血清素：番紅花素與番紅花醛被關聯到影響血清素系統，這是抗憂鬱與情緒研究引用的機制基礎。 <a href="#ref-6">[6]</a> 其二是與 GABA 系統的交互作用，可能與放鬆、入睡有關，屬機制假說層次。 <a href="#ref-2">[2]</a> 其三，Lopresti 團隊 2021 年、針對睡眠品質差成人的三組隨機對照試驗（安慰劑／14 毫克／28 毫克，28 天）觀察到，睡前服用與傍晚唾液褪黑激素上升有關，且不影響皮質醇。 <a href="#ref-3">[3]</a> 這一點是番紅花與直接補充外源褪黑激素在概念上的分野：前者被關聯到提高內生分泌，後者是外加。要提醒的是，以上多屬機制與生化層級的觀察，距離「治好失眠或憂鬱」還有一段路，不能直接畫上等號。

## 食物來源／常見來源

番紅花在飲食中的角色是香料，用於燉飯、湯品與甜點的染色與調香，一般烹調用量極少。保健研究使用的劑量（每日約 28–30 毫克標準化萃取）遠高於日常入菜的量，靠飲食難以達到研究劑量，因此它在實務上屬於補充品議題而非飲食來源議題。 <a href="#ref-1">[1]</a> 純番紅花香料單價高，市面也常見以紅花（safflower）、薑黃等廉價材料混充或染色的情形；選購香料與選購標準化保健原料是兩回事。

## 建議攝取量／常見攝取建議／研究中常見使用方式

番紅花萃取沒有每日必需攝取量。人體試驗最常見的助眠與情緒劑量集中在每日 28–30 毫克標準化萃取：助眠試驗多用 affron 每日 28 毫克（14 毫克一日兩次）、療程約 4 週。 <a href="#ref-1">[1]</a> 情緒試驗給出的劑量門檻很具體——每日 28 毫克有效、22 毫克在同一項研究中未見治療效果。 <a href="#ref-4">[4]</a> 抗憂鬱比較類的試驗常用劑量約每日 30 毫克、療程約 6 週。 <a href="#ref-5">[5]</a> 要點是：現有正面結果集中在這個劑量帶，低於此劑量的效果證據薄弱，而高劑量並不會更好，且會逼近安全上限（見安全性一節）。台灣藥師與品牌衛教常見的建議是每日 14–30 毫克、早晚分兩次。 <a href="#ref-10">[10]</a>

## 缺乏風險或攝取不足問題

番紅花萃取不是必需營養素，沒有「缺乏症」。不補充它不會造成任何已知健康缺損。把失眠或情緒低落直接歸因於「缺番紅花」並不合理——睡眠品質更受作息、光照、咖啡因與壓力影響，情緒困擾則牽涉生活、人際與可能的疾病因素。它的定位是可選的機能性補充，而不是「缺了要補」。真的長期睡不好或情緒困擾，優先做的是尋求醫療與專業評估，而不是自行加補充品。

## 補充品常見形式

市售多為標準化番紅花萃取的膠囊或錠劑，常見標定番紅花素與番紅花醛含量，並以 affron 等專利原料為賣點。 <a href="#ref-1">[1]</a> 也常與 GABA、色胺酸、鎂、維生素 B 群等做成助眠或情緒複方。複方看似完整，卻讓判讀變難：有效或不適都不易歸因到單一成分。選購時可看每份的標準化萃取毫克數（對照研究劑量帶 28–30 毫克）、是否標定活性成分、是否為研究採用的原料，以及有無第三方檢驗，而非被「天然安眠」「媲美抗憂鬱藥」這類超前證據的文案主導。

## 安全性與注意事項

保健劑量的番紅花耐受度良好：在 Lopresti 團隊 2020、2021 年與 Kell 團隊 2017 年等安慰劑對照試驗中，每日約 28–30 毫克標準化萃取的不良事件少於或接近安慰劑組。 <a href="#ref-1">[1]</a> <a href="#ref-3">[3]</a> <a href="#ref-4">[4]</a> 與 SSRI 相比，統合分析也觀察到番紅花的不良事件較少。 <a href="#ref-5">[5]</a> 但劑量安全有明確邊界：毒理學回顧整理，番紅花約每日 1.5 公克以內視為相對安全，約 5 公克出現毒性反應與子宮刺激／催經作用，約 20 公克可致命，動物高劑量還顯示致畸性。 <a href="#ref-7">[7]</a> 研究用的 28–30 毫克遠低於這些門檻，但這也說明自行大幅加量沒有意義且有風險。特別注意族群與交互作用：孕婦與哺乳者屬明確禁忌；出血傾向或服用抗凝血劑者、服用抗憂鬱藥（尤其 SSRI，理論上血清素作用可能疊加）者、雙相情緒疾患者，使用前應先諮詢醫師或藥師。有睡眠或情緒困擾時，請以專業醫療評估為優先，本文僅整理機制與整體人體證據，不對個別讀者做療效或替代處方的判斷。

## 常見迷思

- 「番紅花是天然安眠藥」：它的助眠證據來自數十人規模、4 週內的短期試驗，終點是主觀睡眠量表（如 ISI、PSQI），且部分試驗由原料相關團隊執行，把它當成安眠藥的替代並不成立。 <a href="#ref-1">[1]</a> 台灣把褪黑激素列為藥品、須醫師處方，番紅花則以保健食品身分販售，兩者的管理與定位不同，褪黑激素也不宜自行從境外購買。 <a href="#ref-8">[8]</a>
- 「效果媲美抗憂鬱藥」：2024 年一篇彙整 12 篇隨機對照試驗的統合分析指出，番紅花在憂鬱、焦慮改善上與 SSRI 無顯著差異、不良事件較少；但同一分析也點出樣本小、療程短、最佳劑量與療程未定，屬中低證據，不宜當成停藥或替代處方藥的依據。 <a href="#ref-5">[5]</a>
- 「劑量越高越有效」：情緒試驗中每日 22 毫克無效、28 毫克才有效，顯示存在劑量門檻；但高劑量會逼近毒性上限，多補沒有額外好處。 <a href="#ref-4">[4]</a>
- 「孕婦也能安心吃」：高劑量番紅花具子宮刺激與催經作用、動物高劑量有致畸性，孕婦與哺乳者屬明確禁忌族群；即使保健劑量遠低於毒性門檻，仍建議這兩個族群避免。 <a href="#ref-7">[7]</a>
- 「跟吃褪黑激素一樣」：機制上番紅花被關聯到提高傍晚內生褪黑激素分泌並調節血清素、GABA，屬機制訊號，與直接補充外源褪黑激素在概念上不同；2025 年一篇針對中度失眠者的去中心化隨機對照試驗延續了小規模、短療程的證據型態，整體仍需更大規模與台灣本地族群的資料。 <a href="#ref-11">[11]</a> <a href="#ref-3">[3]</a>
