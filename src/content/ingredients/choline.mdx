## 30 秒認識

- 膽鹼（Choline）是必需營養素，是構成細胞膜（磷脂醯膽鹼）、神經傳導物質（乙醯膽鹼）與肝臟脂質運輸的原料。 <a href="#ref-1">[1]</a>
- 人體雖能自行合成少量，但通常不足以滿足需求，仍須由飲食取得；**多數人、包括 9 成以上孕婦攝取量低於建議值**。 <a href="#ref-2">[2]</a>
- 它在孕期格外重要：足量攝取與後代的資訊處理速度、持續注意力等指標有關。 <a href="#ref-2">[2]</a>
- 蛋（尤其蛋黃）是最方便的膽鹼來源之一；至於坊間「補腦」保健品（如 α-GPC、citicoline），對健康人的效益證據仍有限。

## 基本介紹

膽鹼在 1998 年才被美國醫學研究院正式列為必需營養素。 <a href="#ref-1">[1]</a> 它常和維生素 B 群一起被討論，但本身不是維生素。身體可在肝臟以磷脂醯乙醇胺合成少量膽鹼，惟通常不足以涵蓋全部需求，因此仍屬「需由飲食補足」的營養素。它的功能橫跨結構（細胞膜）、訊號（神經傳導）與代謝（甲基供體、脂質運輸）三個層面。

## 這是什麼？

標示上常見 Choline，或以其衍生物形式出現，如磷脂醯膽鹼（phosphatidylcholine，又稱卵磷脂的主要成分）、甘油磷脂醯膽鹼（α-GPC）、胞磷膽鹼（citicoline／CDP-choline）等。要先建立一個觀念：**它是日常必需營養素，不是只在補腦保健品裡才有的「聰明成分」**；先把飲食攝取補足，比追逐特定衍生物更基本。

## 在身體裡做什麼？／主要作用機制

膽鹼有三條主要路徑： <a href="#ref-1">[1]</a> 第一，合成磷脂醯膽鹼與鞘磷脂，構成細胞膜並參與極低密度脂蛋白（VLDL）把脂肪運出肝臟——這是它與「脂肪肝」相關的原因；第二，合成神經傳導物質乙醯膽鹼，參與記憶、肌肉控制等；第三，經氧化為甜菜鹼（betaine）作為甲基供體，與葉酸、B12 共同代謝同半胱胺酸。胎兒腦部（尤其海馬迴）發育對膽鹼需求高，這是孕期研究關注的核心。 <a href="#ref-2">[2]</a>

## 食物來源／常見來源

膽鹼以動物性食物含量較高，蛋與肝臟是代表。以 NIH ODS 的代表含量（每份約略值）： <a href="#ref-1">[1]</a>

- 牛肝（煎，約 85 克）：約 356 毫克（含量最高的常見食物之一）
- 雞蛋（大型，1 顆，含蛋黃）：約 147 毫克
- 牛肉、雞肉、鮭魚（約 85 克）：約 70–115 毫克
- 黃豆（炒，半杯）：約 107 毫克
- 馬鈴薯、十字花科蔬菜（綠花椰菜、抱子甘藍）：數十毫克不等
- 小麥胚芽、腰豆：中等含量

值得一提：膽鹼大量集中在蛋黃，只吃蛋白會損失主要來源。對素食者而言，黃豆製品、十字花科蔬菜與全穀是較可行的補充方向，但要吃到孕期需求通常較費力。

## 建議攝取量／常見攝取建議／研究中常見使用方式

膽鹼目前只有「適量攝取量（AI）」而非 RDA，因為資料還不足以訂出 RDA。AI 為： <a href="#ref-1">[1]</a>

- 成年男性：每日 550 毫克；成年女性：425 毫克
- 懷孕：450 毫克；哺乳：550 毫克

要提醒兩點：第一，現行 AI 是 1998 年依「預防成年男性肝損傷」所訂，近年研究認為對孕期可能偏低——一項控制飲食試驗讓孕婦第三孕期攝取 480 或 930 毫克，後代資訊處理速度在較高攝取組較快。 <a href="#ref-2">[2]</a> 第二，現實中多數人攝取不足，光靠單一食物不易達標，雞蛋、瘦肉、黃豆製品分散在各餐是較務實的做法。

## 缺乏風險或攝取不足問題

明顯的膽鹼缺乏會造成肝臟脂肪堆積（非酒精性脂肪肝）與肌肉損傷。 <a href="#ref-1">[1]</a> 風險較高者包括：長期攝取不足者、停經後女性（雌激素能促進內生合成，停經後此保護下降）、某些基因多型者，以及全靜脈營養（TPN）病人。要注意的是，疲勞或記憶力不佳不應直接歸因於「缺膽鹼」；真正的缺乏需由臨床與飲食評估判斷，而非以保健品自我診斷。

## 補充品常見形式

常見酒石酸膽鹼（choline bitartrate）、卵磷脂／磷脂醯膽鹼，以及主打認知的 α-GPC 與 citicoline。 <a href="#ref-1">[1]</a> α-GPC 與 citicoline 在部分臨床族群（如特定神經疾病）有研究，但**用於健康人「增強記憶或專注」的證據仍有限**，行銷常超出實證。對一般人而言，先用天然食物把膽鹼攝取補足，比花錢買高階衍生物更有意義；孕婦若考慮補充，應與醫師或營養師討論劑量與必要性。

## 安全性與注意事項

膽鹼成人每日可耐受上限為 3500 毫克，過量（多來自高劑量補充品而非食物）可能造成魚腥體味、出汗、流口水、腸胃不適與血壓下降。 <a href="#ref-1">[1]</a> 另一個被討論的議題是 **TMAO**：腸道細菌可把膽鹼、肉鹼代謝為三甲胺，再經肝臟氧化為 TMAO，觀察性研究發現 TMAO 偏高與心血管風險有關聯。但這屬關聯性、因果與「是否該因此少吃蛋」仍未定論，且膽鹼在孕期等的明確益處不應因此被忽略。有肝腎疾病或正在用藥者，補充前宜諮詢專業。

## 常見迷思

- 「膽鹼只是補腦保健品成分」：它是日常必需營養素，缺乏會傷肝、影響孕期；補腦衍生物對健康人的效益反而證據有限。 <a href="#ref-1">[1]</a>
- 「吃蛋會升膽固醇所以要少吃」：蛋黃同時是膽鹼的重要來源；對多數人，飲食膽固醇對血脂的影響不如過去認知大，孕期更不宜因此排除蛋黃。 <a href="#ref-1">[1]</a>
- 「補越多越聰明」：超過上限可能造成體味、低血壓等不適；認知增強對健康人並無一致證據。
- 「素食一定缺膽鹼」：黃豆製品、十字花科與全穀也能提供，只是要達孕期需求較需用心安排。
