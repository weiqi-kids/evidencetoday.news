## 30 秒認識

- 亞精胺是一種天然多胺分子，人體細胞、腸道菌與多種發酵或熟成食物皆會產生，小麥胚芽、納豆與熟成乾酪是常見的高含量來源 <a href="#ref-5">[5]</a>。
- 在酵母、線蟲、果蠅與囓齒類的動物實驗中，亞精胺能誘導細胞自噬並延長壽命，是目前抗老研究最常被引用的機制證據 <a href="#ref-5">[5]</a>。
- 義大利布魯內克世代研究追蹤 829 位居民長達 20 年，發現飲食亞精胺攝取量較高的族群全因死亡率較低，美國大型飲食調查資料庫的分析也觀察到類似方向，但這些都只是相關性，不能證明是亞精胺本身造成 <a href="#ref-1">[1]</a><a href="#ref-6">[6]</a>。
- 一項納入 30 人的小型前導試驗曾顯示補充亞精胺對記憶表現有正向訊號，但後續規模擴大到 100 人、追蹤 12 個月的第二期隨機對照試驗並未證實有顯著效益，兩項結果並不一致 <a href="#ref-2">[2]</a><a href="#ref-3">[3]</a>。
- 英國(沿用歐盟核准架構)已將小麥胚芽萃取的亞精胺核准為新穎食品原料，每日核准上限約 6 毫克(排除孕婦、哺乳婦女)，但市售產品劑量差異頗大，選購前宜留意標示與實際含量是否對應這個上限 <a href="#ref-7">[7]</a>。

## 基本介紹

亞精胺(spermidine)屬於多胺(polyamine)家族的一員，與腐胺(putrescine)、精胺(spermine)同屬一類含有多個胺基的小分子。多胺廣泛存在於所有生物的細胞內，參與 DNA 穩定、細胞分裂與蛋白質合成等基本生理功能，並非近年才被發現的新成分，只是近十多年因細胞自噬研究興起而重新受到矚目 <a href="#ref-5">[5]</a>。人體本身會透過胺基酸代謝合成亞精胺，腸道菌相也會產生一部分，飲食則是額外補充的來源之一。

## 這是什麼?

化學上，亞精胺是一種含有三個胺基的直鏈脂肪族多胺，分子量小、水溶性高，廣泛分布於動植物、真菌與細菌體內。它在細胞中的濃度會隨年齡增加而逐漸下降，這個現象在多個物種的研究中都被觀察到，也是「亞精胺與老化有關」這個假說的起點之一 <a href="#ref-5">[5]</a>。需要留意的是，細胞內濃度下降與「補充亞精胺就能逆轉老化」是兩件不同的事，前者是觀察到的現象，後者則是尚待更多人體證據支持的推論。

## 在身體裡做什麼?/主要作用機制

亞精胺目前最受關注的機制是誘導細胞自噬(autophagy)——細胞清除受損蛋白質與胞器、回收再利用的自我清潔機制。在酵母、線蟲、果蠅與小鼠的實驗中，研究者觀察到亞精胺能活化自噬相關基因，並延長這些生物的壽命；若透過基因方式關閉自噬機制，亞精胺的延壽效果就會消失，顯示自噬確實是其中一個關鍵路徑 <a href="#ref-5">[5]</a>。除了自噬之外，亞精胺也被觀察到與抑制發炎反應、影響脂質代謝、參與心血管保護等機制有關，大型飲食世代分析也發現攝取量與心血管疾病風險呈負向關聯，但這些機制多數仍以動物或細胞實驗、觀察性資料為主，人體層級的因果關係尚未完全確立 <a href="#ref-1">[1]</a><a href="#ref-6">[6]</a>。

## 食物來源/常見來源

亞精胺普遍存在於植物性與發酵食物中，其中小麥胚芽是常被文獻引用的高含量來源之一；納豆等大豆發酵食品，以及熟成時間較長的乾酪(如切達、帕瑪森),在發酵或熟成過程中多胺濃度也會提高。此外，菇類、豆科植物與部分堅果類也含有一定量的亞精胺 <a href="#ref-5">[5]</a><a href="#ref-1">[1]</a>。由於不同產地、品種與加工方式都會影響實際含量，坊間流傳的「每 100 公克含量」數字時常互有出入，建議把這些食物理解為「相對含量較高的類別」,而非精準到毫克的營養標示。

## 建議攝取量/常見攝取建議/研究中常見使用方式

目前並沒有官方訂定的每日建議攝取量(RDA)。人體研究中使用的劑量差異相當大:記憶相關的小型前導試驗與後續大型試驗多使用每日約 1 毫克上下、透過富含亞精胺的植物萃取物補充 <a href="#ref-2">[2]</a><a href="#ref-3">[3]</a>;而以安全性為主的探索性試驗則測試過每日 40 毫克、連續補充最長 28 天，觀察高純度亞精胺對健康年長男性的耐受度 <a href="#ref-4">[4]</a>。換言之，「多少劑量有效」在人體研究中尚未收斂到一致答案，市售產品標示的劑量也未必對應到已有臨床試驗驗證過的用量，選購前應理解這一點。

## 缺乏風險或攝取不足問題

目前沒有明確定義的「亞精胺缺乏症」,也沒有證據顯示健康成人若飲食中亞精胺攝取偏低會導致特定疾病。細胞內亞精胺濃度隨年齡下降是普遍觀察到的生理現象，但這與臨床上定義的營養素缺乏不同，不宜直接類比為缺乏維生素或礦物質那樣需要積極「補足」的狀態 <a href="#ref-5">[5]</a>。

## 補充品常見形式

市面上的亞精胺補充品主要有兩種來源:一種是從小麥胚芽萃取、保留天然多胺組合的植物萃取物，英國(沿用歐盟核准架構)已將這類小麥胚芽萃取物核准為新穎食品原料，每日核准上限約 6 毫克(排除孕婦、哺乳婦女)；另一種是高純度化學合成的亞精胺三鹽酸鹽(spermidine trihydrochloride),常用於劑量較高的探索性安全試驗 <a href="#ref-4">[4]</a><a href="#ref-7">[7]</a>。兩者的實際生體可用率、與食物基質共同攝取時的吸收情形，目前公開的人體資料仍有限，消費者選購時很難單憑成分名稱判斷實際效果差異。

## 安全性與注意事項

已發表的探索性安全性試驗顯示，健康年長男性每日補充 40 毫克高純度亞精胺、連續最長 28 天，並未觀察到明顯的臨床、血脂或血液學異常，耐受度良好；但研究也發現這個劑量下血液與尿液中的多胺濃度變化並不明顯，顯示人體對外來亞精胺可能有一定程度的恆定調節機制 <a href="#ref-4">[4]</a>。這類試驗樣本數小、追蹤時間短，長期、大劑量使用的安全性資料仍不完整。孕婦、哺乳婦女、慢性病患者或服藥中人士，若考慮額外補充，建議先與醫療專業人員討論，而非僅參考補充品廠商的行銷資訊。

## 常見迷思

- **迷思:「亞精胺是抗老聖品，吃了就能逆轉老化」。** 目前支持這個說法的主要是動物實驗與觀察性世代研究，尚無足夠大型、長期的人體隨機對照試驗證實補充亞精胺能延長壽命或逆轉老化 <a href="#ref-5">[5]</a>。
- **迷思:「補充亞精胺一定能改善記憶力」。** 小型前導試驗確實出現正向訊號，但規模更大、追蹤更久的後續試驗並未重現這個效果，兩者結果不一致，目前不宜當作定論 <a href="#ref-2">[2]</a><a href="#ref-3">[3]</a>。
- **迷思:「天然食物來源的亞精胺沒有劑量問題，越多越好」。** 目前多數人體證據來自中低劑量的飲食觀察或小劑量補充試驗，高劑量長期使用的安全性資料仍有限，並非「天然來源」就等於沒有上限 <a href="#ref-4">[4]</a>。
- **迷思:「布魯內克研究證明亞精胺能降低死亡率」。** 該研究是觀察性世代分析，只能呈現攝取量與死亡率之間的相關性，無法排除飲食型態、生活習慣等干擾因素，不能直接推論為因果關係 <a href="#ref-1">[1]</a>。
