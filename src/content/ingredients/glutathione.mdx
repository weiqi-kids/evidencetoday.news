## 30 秒認識

- 穀胱甘肽（GSH）是由麩胺酸、半胱胺酸、甘胺酸組成的三胜肽，人體肝臟等細胞可自行合成，本身就是體內主要的抗氧化與解毒系統之一 <a href="#ref-3">[3]</a>。
- 亞洲市場常見的「口服／注射美白」訴求，核心爭議在於口服穀胱甘肽能否真正提高體內或皮膚中的濃度，這一點學界長年沒有定論 <a href="#ref-4">[4]</a>。
- 多篇隨機對照試驗測得口服後黑色素指數略降，但多數樣本小、追蹤僅8至12週，部分結果未達統計顯著 <a href="#ref-1">[1]</a><a href="#ref-2">[2]</a>。
- 靜脈注射「美白針」僅有少數小型試驗（如收錄於系統性回顧中的安慰劑對照試驗）評估效果，結果顯示短期改善不持久、且伴隨較高比例肝功能異常等不良反應，仍缺乏能確立安全劑量的大規模長期臨床試驗，已知風險包括肝腎負擔與嚴重過敏反應，多國衛生主管機關發布警告 <a href="#ref-2">[2]</a><a href="#ref-6">[6]</a><a href="#ref-8">[8]</a>。
- 台灣將穀胱甘肽列為可用食品原料，但明訂每日食用限量250毫克，並要求包裝標示過敏、孕婦及嬰幼兒的注意事項 <a href="#ref-7">[7]</a>。

## 基本介紹

穀胱甘肽並非需要額外攝取才會出現在人體內的物質，而是幾乎每個細胞都能自行合成的胜肽，在肝臟中濃度特別高，長期被視為體內抗氧化防禦網的核心成員之一 <a href="#ref-3">[3]</a>。近十年市場出現大量口服錠劑、發泡錠與靜脈注射產品，主打「美白」「淡斑」，但這類商業訴求與學術文獻對它效果的評價，落差相當明顯。

## 這是什麼？

穀胱甘肽是由麩胺酸（glutamate）、半胱胺酸（cysteine）、甘胺酸（glycine）三個胺基酸組成的三胜肽，化學結構簡單但生理角色多元，同時參與抗氧化反應、藥物與毒素代謝、免疫細胞功能調節 <a href="#ref-3">[3]</a>。市售產品多以「圓酵母（Torula yeast）發酵」方式生產原料，這也是台灣食品法規中核准的主要來源型態 <a href="#ref-7">[7]</a>。

## 在身體裡做什麼？／主要作用機制

穀胱甘肽的核心功能是清除細胞內的自由基與過氧化物，並協助肝臟進行第二階段解毒反應，將部分毒素或藥物代謝產物接合後排出體外 <a href="#ref-3">[3]</a>。在美白相關的討論中，主流假說是穀胱甘肽可能抑制酪胺酸酶活性、並促使黑色素生成路徑從深色的真黑色素（eumelanin）轉向較淺色的褐黑色素（pheomelanin），但這個機制多來自體外或動物實驗，人體皮膚內是否以同樣路徑發生、需要多少劑量與多久時間才會反映在膚色上，仍缺乏一致的活體證據 <a href="#ref-2">[2]</a><a href="#ref-4">[4]</a>。

## 食物來源／常見來源

天然食物中，穀胱甘肽本身較不穩定、烹煮易分解，因此多數飲食建議轉為著重提供其原料胺基酸，尤其是含硫的半胱胺酸。富含半胱胺酸的食物包括雞肉、火雞肉、蛋、大蒜、優格、黃豆與部分豆科食物，人體攝取這些原料後，仍需依賴自身代謝機制合成穀胱甘肽，並非直接吸收現成的穀胱甘肽分子 <a href="#ref-3">[3]</a>。

## 建議攝取量／常見攝取建議／研究中常見使用方式

口服美白相關研究中最常見的劑量落在每日250毫克至500毫克，療程多為8至12週 <a href="#ref-1">[1]</a><a href="#ref-2">[2]</a>。另一派探討「能否提升體內穀胱甘肽儲備」的研究，則使用較高劑量：一項連續6個月的隨機對照試驗同時測試每日250毫克與1000毫克兩組，結果兩組受試者紅血球與淋巴球中的穀胱甘肽濃度皆顯著上升、僅上升幅度不同，但這是否等同於皮膚美白效果，該研究並未評估 <a href="#ref-5">[5]</a>。台灣食品法規將以圓酵母發酵製成的穀胱甘肽，每日食用上限訂為250毫克 <a href="#ref-7">[7]</a>，等於市面常見的高劑量產品已逼近或超過法規上限。

## 缺乏風險或攝取不足問題

穀胱甘肽屬於人體自行合成的物質，目前沒有公認的「缺乏症」診斷標準，一般健康成人不需要額外補充來維持基本生理功能 <a href="#ref-3">[3]</a>。部分慢性發炎或代謝疾病患者血中穀胱甘肽濃度較低已在觀察性研究中被提及，但這類關聯屬於相關性描述，並不代表補充穀胱甘肽能改善或逆轉疾病本身 <a href="#ref-3">[3]</a>。

## 補充品常見形式

市面產品主要分為口服（膠囊、錠劑、發泡錠、口含錠）與注射（靜脈注射，俗稱「美白針」）兩大類。口服劑型近年也出現複方設計，常與維生素C、硫辛酸（alpha-lipoic acid）、鋅併用，理論上是希望透過抗氧化協同作用提升效果 <a href="#ref-1">[1]</a>。口含錠、舌下錠訴求可繞過腸道分解、提高吸收率，但相關人體證據仍屬初步、樣本有限 <a href="#ref-4">[4]</a>。

## 安全性與注意事項

口服穀胱甘肽在現有研究中多半耐受性良好，副作用集中在輕微腹脹、噁心等腸胃道不適 <a href="#ref-1">[1]</a>。靜脈注射則風險層級明顯不同：僅有少數小型試驗（如收錄於系統性回顧中的Zubair等2016年安慰劑對照試驗）評估其美白效果，結果顯示短期效果不持久、且伴隨較高比例的肝功能異常等不良反應，仍缺乏能確立安全劑量的大規模長期臨床試驗；已知的不良反應包括肝腎功能負擔，嚴重時可能誘發史蒂文生－強生症候群（Stevens-Johnson syndrome）等致命過敏反應，若在非醫療院所以非無菌方式注射，還有血液感染風險 <a href="#ref-2">[2]</a><a href="#ref-3">[3]</a><a href="#ref-6">[6]</a><a href="#ref-8">[8]</a>。菲律賓食品藥物管理局已針對此類產品發布安全警告 <a href="#ref-6">[6]</a>；台灣衛生福利部亦曾說明「美白針」並非核准上市的美白適應症用藥，屬於藥品的仿單外使用，需留意藥物交互作用與過敏反應 <a href="#ref-9">[9]</a>。孕婦、哺乳婦女、嬰幼兒及對穀胱甘肽過敏者，依台灣食品法規標示規定應避免食用含穀胱甘肽的食品 <a href="#ref-7">[7]</a>。

## 常見迷思

- **迷思：「口服穀胱甘肽一定會被腸胃分解、完全沒用」**。這個說法源自1990年代的經典研究，發現單次高劑量口服後血中濃度幾乎沒有變化 <a href="#ref-4">[4]</a>；但後續一項連續數月、每日補充的試驗則測得體內儲備可能緩慢提升，兩者結論不同，說明「有沒有效」需要區分單次劑量與長期補充兩種情境 <a href="#ref-5">[5]</a>。
- **迷思：「穀胱甘肽美白效果已獲科學證實」**。系統性回顧指出現有臨床試驗數量少、樣本規模小、追蹤期短，且納入研究的風險偏差（risk of bias）評估參半、高風險與低風險研究約各占一半；雖口服試驗合併分析顯示黑色素指數有統計上顯著降低，效果證據仍屬初步，仍需更大規模、更長期的隨機對照試驗才能做出明確療效宣稱 <a href="#ref-2">[2]</a>。
- **迷思：「美白針比口服更快更有效，所以更值得選」**。速度與效果無關安全性：靜脈注射目前僅有少數小型試驗評估效果（如安慰劑對照試驗顯示短期改善不持久、且伴隨較高比例肝功能異常等不良反應），仍缺乏能確立安全劑量的大規模長期臨床試驗，風險評估反而是各劑型中最不明確的一種 <a href="#ref-2">[2]</a><a href="#ref-6">[6]</a><a href="#ref-8">[8]</a>。
- **迷思：「天然抗氧化物，吃越多越安全」**。穀胱甘肽在台灣屬於有每日限量規定的食品原料，過量攝取並非「有益無害」，仍應留意標示上限與個人體質警語 <a href="#ref-7">[7]</a>。
