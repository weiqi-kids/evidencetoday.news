## 30 秒認識

- 蘿蔔硫素（Sulforaphane）是十字花科蔬菜中的異硫氰酸酯，本身不直接存在於植物裡，而是以前驅物「蘿蔔硫苷（glucoraphanin）」儲存，需黑芥子酶（myrosinase）水解才生成。 <a href="#ref-1">[1]</a>
- 濃縮來源是青花椰菜「苗」：3 日齡青花椰菜苗的前驅物 glucoraphanin 含量可達成熟花椰菜的 10–100 倍（Fahey 等人 1997 年發表於 PNAS 的經典測定）。 <a href="#ref-2">[2]</a>
- 生理上它是強效的天然 Nrf2 活化劑之一：修飾 Keap1 蛋白的半胱胺酸殘基，讓轉錄因子 Nrf2 進入細胞核，上調 NQO1、麩胱甘肽 S-轉移酶（GST）、血基質氧化酶-1（HO-1）等第二相解毒與抗氧化酵素——是啟動身體自身的酵素防禦，而非直接清除自由基。 <a href="#ref-3">[3]</a>
- 高溫（如水煮）會使黑芥子酶失活，大幅降低蘿蔔硫素的吸收——一項人體交叉試驗中，水煮花椰菜的蘿蔔硫素生體可用率僅約 3.4%，遠低於生食的 37%。 <a href="#ref-4">[4]</a> 補救方式是生食、輕蒸，或在煮熟的花椰菜額外拌入芥末（芥菜籽）粉、白蘿蔔泥等外源酵素：一項隨機交叉試驗發現，補充芥菜籽 myrosinase 可讓蘿蔔硫素吸收約增加一倍（18.6% → 39.8%）。 <a href="#ref-5">[5]</a>
- 人體效益要分開誠實看：肝指數（ALT）、胰島素阻抗（HOMA-IR，見於第二型糖尿病患者）各有小型隨機對照試驗看到改善，但 2025 年一項較大型的糖尿病前期試驗未達主要終點，空腹血糖僅降約 0.2 mmol/L——機制看好，人體證據仍多為小型、短期、以生化指標為主。 <a href="#ref-6">[6]</a> <a href="#ref-7">[7]</a> <a href="#ref-8">[8]</a>

## 基本介紹

蘿蔔硫素是十字花科蔬菜特有的含硫植化素（異硫氰酸酯的一種）。它在完整蔬菜細胞裡以穩定的前驅物 glucoraphanin 形式存在，與黑芥子酶（myrosinase）分處不同區隔；當蔬菜被切碎、咀嚼，或苗被壓破，酵素才與前驅物接觸、水解生成蘿蔔硫素。 <a href="#ref-1">[1]</a> 保健品市場上的產品，有的直接標示「sulforaphane」，有的標示前驅物「glucoraphanin」含量，兩者不能畫上等號——含 glucoraphanin 不代表身體真的吸收到等量蘿蔔硫素，關鍵在於有沒有活性酵素幫忙轉換。 <a href="#ref-5">[5]</a>

## 這是什麼？

標示上常見 Sulforaphane（蘿蔔硫素、蘿蔔硫烷）。先建立兩個觀念：第一，它是植物性的抗氧化與解毒酵素誘導物，而非人體必需營養素，沒有每日必需量；第二，它的價值來自「前驅物加酵素」這組配套——青花椰菜苗之所以是濃縮來源，是因為苗期同時富含高濃度 glucoraphanin。3 日齡青花椰菜苗的 glucoraphanin 含量約為成熟花椰菜的 10–100 倍，實際倍數隨品種、苗齡與檢測方法而有差距。 <a href="#ref-1">[1]</a> <a href="#ref-2">[2]</a>

## 在身體裡做什麼？／主要作用機制

蘿蔔硫素最被研究的機制是活化 Nrf2–Keap1 抗氧化路徑。平時 Nrf2 被 Keap1 綁住、送去分解；蘿蔔硫素會修飾 Keap1 上的半胱胺酸殘基，使 Nrf2 得以釋出、進入細胞核並結合抗氧化反應元件（ARE），進而上調 NQO1、GST、HO-1、γ-穀胺醯半胱胺酸合成酶等第二相解毒與抗氧化酵素。 <a href="#ref-3">[3]</a> 中國農業大學團隊 2024 年在《Nutrients》的回顧文章，整理了這條路徑，以及蘿蔔硫素對肝臟葡萄糖攝取、脂肪酸氧化與胰島素訊號的動物與細胞層級作用。 <a href="#ref-3">[3]</a> 要提醒的是：這些多屬機制與動物、細胞層級的觀察，指的是身體可能被誘導出更強的自身防禦，而非直接證明能治療特定疾病；把機制推論成療效，目前人體證據還接不上。

## 食物來源／常見來源

蘿蔔硫素的來源是十字花科蔬菜：青花菜（綠花椰）、白花椰菜、高麗菜、芥藍、白蘿蔔等都含有其前驅物，其中以青花椰菜苗最濃縮。 <a href="#ref-1">[1]</a> 台灣讀者熟悉的「青花椰菜苗／綠花椰苗」多是 3–5 天的嫩苗，正是 glucoraphanin 的高峰期。 <a href="#ref-2">[2]</a> 實務上有兩個關鍵陷阱：一是酵素——高溫烹調會讓黑芥子酶失活，煮熟的花椰菜若沒有外源酵素幫忙，蘿蔔硫素生成與吸收會大打折扣； <a href="#ref-4">[4]</a> 二是吸收——沒有活性酵素的萃取物，人體吸收顯著低於新鮮苗。台灣家庭常見的做法可以派上用場：花椰菜洗切後靜置數分鐘再輕蒸，或起鍋後拌入少量黃芥末、芥菜籽粉或白蘿蔔泥補充外源酵素。 <a href="#ref-5">[5]</a>

## 建議攝取量／常見攝取建議／研究中常見使用方式

蘿蔔硫素沒有官方每日必需攝取量。人體試驗的用量落差很大，且劑型不一，難以直接互相換算：肝功能試驗用的是標準化 glucoraphanin 膠囊，每日約 60 毫克（137.1 μmol）、持續 24 週； <a href="#ref-6">[6]</a> 胰島素阻抗試驗用的是青花椰菜苗粉，第二型糖尿病患者每日 10 公克、持續 4 週。 <a href="#ref-7">[7]</a> 這裡要特別分清楚：常被引用的「10–100 倍」指的是前驅物 glucoraphanin，而非成品蘿蔔硫素本身，且會隨品種、苗齡、儲存與烹調大幅變動，不宜寫成單一絕對值。 <a href="#ref-1">[1]</a> <a href="#ref-2">[2]</a> 也因為吃進 glucoraphanin 不等於吸收到 sulforaphane，不能把試驗劑量直接換算成「每天要吃幾份花椰菜」。 <a href="#ref-5">[5]</a>

## 缺乏風險或攝取不足問題

蘿蔔硫素不是必需營養素，沒有「缺乏症」，不吃它並不會造成已知的健康缺損。 <a href="#ref-1">[1]</a> 因此合理的定位是「十字花科蔬菜整體飲食的一部分」，而非「缺了就要補」。與其追逐單一成分，均衡攝取各色蔬菜（含十字花科）本身就有公認的營養價值；把疲倦、代謝或老化問題直接歸因於「缺蘿蔔硫素」並不合理，這些更受整體飲食、作息與既有疾病影響。

## 補充品常見形式

市售產品主要有三類：新鮮青花椰菜苗、乾燥苗粉，以及青花椰菜（苗或籽）萃取的膠囊、粉末。 <a href="#ref-1">[1]</a> 挑選時有兩個重點：一是看標示的是「glucoraphanin（前驅物）」還是「sulforaphane（成品）」，以及每份劑量；二是萃取物是否含有活性 myrosinase，或建議搭配外源酵素——缺乏活性酵素的萃取物，蘿蔔硫素吸收會明顯較低。 <a href="#ref-5">[5]</a> 複方產品（常與薑黃、綠茶萃取等組合）會讓效果與不適都更難歸因，判讀時要多留意。

## 安全性與注意事項

在既有的人體試驗中，青花椰菜苗與其萃取物多被受試者良好耐受：肝功能試驗以每日 60 毫克 glucoraphanin 連續 24 週，依從率超過 99%； <a href="#ref-6">[6]</a> 胰島素阻抗試驗以第二型糖尿病患者每日 10 公克苗粉連續 4 週。 <a href="#ref-7">[7]</a> 常見的注意事項包括：部分人食用十字花科蔬菜或苗粉後可能有脹氣等腸胃不適；正在服用降血糖或影響血糖藥物者，若要規律大量食用，宜與醫療專業討論並自我監測。 <a href="#ref-7">[7]</a> 服用 warfarin（華法林）等抗凝血劑者，因十字花科蔬菜含維生素 K、可能影響 INR，宜維持攝取量穩定並與處方醫療專業討論（青花椰菜苗維生素 K 相對較低，一般攝取量風險有限）；碘攝取不足者若長期大量生食十字花科，其中的致甲狀腺腫物質理論上可能影響甲狀腺，一般飲食量與熟食下屬低度風險。生鮮芽菜（含青花椰菜苗）生食有衛生指標菌風險——衛生福利部提醒，生鮮芽菜清水浸泡 15 分鐘後再漂洗 5 次，可大幅降低微生物含量。 <a href="#ref-10">[10]</a> 本文為成分衛教，不提供個別療效或醫療建議；糖尿病、肝病或正在服藥者，調整飲食或保健品前請先諮詢醫師或營養師，切勿以青花椰菜苗取代處方治療。

## 常見迷思

- 「蘿蔔硫素是抗癌神器、能排毒」：台灣媒體與行銷常見這類框架， <a href="#ref-11">[11]</a> 但目前人體證據多為小型、短期、以生化指標為終點的試驗，不足以支持個別療效宣稱；機制看好不等於臨床療效。 <a href="#ref-9">[9]</a>
- 「多吃煮熟的花椰菜就能補到蘿蔔硫素」：高溫會讓黑芥子酶失活，煮熟又沒有外源酵素時，蘿蔔硫素生成與吸收會大幅減少；生食、輕蒸或補芥末粉、白蘿蔔泥較能保留。 <a href="#ref-4">[4]</a> <a href="#ref-5">[5]</a>
- 「膠囊標示 glucoraphanin 很高就等於補到很多蘿蔔硫素」：含前驅物不等於吸收到成品，缺乏活性酵素的萃取物吸收明顯較低。 <a href="#ref-5">[5]</a>
- 「吃青花椰苗可以降血糖、取代糖尿病藥」：2025 年糖尿病前期試驗未達主要終點，空腹血糖整體僅降約 0.2 mmol/L，僅特定次群體較明顯；不能取代處方治療。 <a href="#ref-8">[8]</a>
