## 30 秒認識

- 納豆激酶（Nattokinase）是納豆以枯草桿菌（Bacillus subtilis natto）發酵大豆時分泌的一種蛋白酶，在試管與動物模型中具有分解纖維蛋白（fibrinolytic，纖溶）的活性，這是它最常被討論的機制。 <a href="#ref-6">[6]</a>
- 機制清楚，人體療效要分開看：目前人體試驗多為小樣本、短期，且以血壓、凝血因子、頸動脈厚度等替代指標為終點，而非心肌梗塞或中風等硬終點。 <a href="#ref-4">[4]</a>
- 血壓方面，Kim 等 2008 年的隨機雙盲安慰劑對照試驗在 86 名血壓偏高成人觀察到收縮壓小幅下降（相較對照組淨變化約 −5.55 mmHg），但屬單一小型試驗。 <a href="#ref-1">[1]</a> 一篇 2023 年、納入 6 篇 RCT 共 546 人的統合分析也發現血壓有統計上顯著但幅度不大的下降。 <a href="#ref-2">[2]</a>
- 最嚴謹的長期試驗（NAPS，Hodis 等 2021，265 人、追蹤約 3 年）發現，納豆激酶對亞臨床動脈硬化與血壓與安慰劑沒有顯著差異，對常見的「溶解斑塊、清血管」宣稱形成反證。值得注意的是，血壓下降的訊號主要出現在血壓偏高的族群，NAPS 收的則是心血管風險較低、血壓多在正常範圍的中高齡受試者，這或可解釋兩組結果為何看似分歧。 <a href="#ref-4">[4]</a>
- 安全最需留意：它以纖溶為主、方向上帶輕度抗凝作用，與抗凝血藥（如 warfarin）或抗血小板藥（如 aspirin、clopidogrel）併用可能疊加出血風險，用藥或有疾病者請先諮詢醫師與藥師。 <a href="#ref-6">[6]</a> <a href="#ref-7">[7]</a>

## 基本介紹

納豆激酶是從日本傳統發酵食品「納豆」分離出來的一種蛋白酶，由枯草桿菌（Bacillus subtilis natto）在發酵大豆的過程中產生。 <a href="#ref-6">[6]</a> 保健食品中的納豆激酶多為萃取、純化後製成的膠囊或錠劑，市面常以「FU」（纖維蛋白溶解／降解活性單位）標示活性強弱；FU 反映的是體外分解纖維蛋白的能力，並非藥品的標準化單位。 <a href="#ref-11">[11]</a>

## 這是什麼？

標示上通常寫 Nattokinase，中文作納豆激酶。要先建立三個觀念：第一，它被歸類為「食品或保健食品成分」而非藥品，依規不得宣稱療效或醫療效能；第二，納豆激酶與納豆不能畫上等號——傳統納豆同時含有豐富的維生素 K2，會影響 warfarin 這類抗凝血藥的作用，萃取型的納豆激酶產品 K2 含量通常較低（依產品與製程而異），但使用前仍應留意； <a href="#ref-8">[8]</a> 第三，它的活性以 FU 標示，各家產品換算基準不一，數字之間不容易直接比較。 <a href="#ref-11">[11]</a>

## 在身體裡做什麼？／主要作用機制

納豆激酶被研究的核心，是它在試管與動物實驗中能直接分解纖維蛋白（fibrin）——纖維蛋白是血液凝固時構成血栓骨架的蛋白質，這也是它被稱為「纖溶酵素」的原因，也是機制上較明確的部分。 <a href="#ref-6">[6]</a> 在人體方面，Hsia 等 2009 年一項開放標籤、自身對照的試驗另觀察到，口服後血漿中的纖維蛋白原與凝血因子 VII、VIII 濃度下降；但這是在沒有安慰劑對照下觀察到的凝血相關生化指標變化，方向上與纖溶／輕度抗凝一致，血脂則未見改變。 <a href="#ref-3">[3]</a> 這些屬於機制與替代指標層面的觀察，與「能預防或治療心血管疾病」之間還有很大距離：把「凝血指標變化」直接推論成「保護心血管」，目前證據並不足夠。

## 食物來源／常見來源

納豆激酶天然存在於日本傳統發酵食品「納豆」中，也就是以枯草桿菌發酵的大豆。 <a href="#ref-6">[6]</a> 依業界常見的換算，補充品標示的每日約 2,000 FU 大約相當於食用 50–100 克納豆的活性量。 <a href="#ref-11">[11]</a> 不過納豆的風味與氣味並非人人能接受，且傳統納豆同時含有較高的維生素 K2，對正在服用抗凝血藥者需另作考量，這也是它在實務上偏向以萃取型補充品討論的原因。真正吃納豆的整體價值，更多來自大豆本身的蛋白質、纖維與發酵帶來的營養，而非單一的納豆激酶。

## 建議攝取量／常見攝取建議／研究中常見使用方式

納豆激酶不是必需營養素，沒有每日必需攝取量。臨床研究中常見的用量差異很大：Kim 等 2008 年的血壓試驗使用每日 2,000 FU、持續 8 週； <a href="#ref-1">[1]</a> Hsia 等 2009 年的凝血指標研究使用每日 4,000 FU、持續 2 個月； <a href="#ref-3">[3]</a> NAPS 長期試驗則為每日 2,000 FU、追蹤約 3 年。 <a href="#ref-4">[4]</a> 日本相關協會常見的每日建議約落在 2,000–4,000 FU 區間。 <a href="#ref-11">[11]</a> 要提醒的是，FU 是體外活性單位，數字越高不代表臨床效果越好，也不適合用來互相比較不同產品；用量與是否適合使用，牽涉個人用藥與健康狀況，屬於應與醫師或藥師討論的範圍。

## 缺乏風險或攝取不足問題

納豆激酶不是人體必需的營養素，因此不存在「缺乏症」，不吃它並不會造成任何已知的健康缺損。 <a href="#ref-6">[6]</a> 它的定位比較接近「可選的保健成分」，而不是「缺了就要補」。從受眾來看，會關注它的多是重視血管與循環保養的中高齡族群； <a href="#ref-8">[8]</a> 但把血壓偏高、血脂異常或循環不佳直接歸因於「缺納豆激酶」並不合理——這些狀況更受飲食型態、運動、體重、抽菸與既有慢性病影響，若有相關疑慮，應交由醫療專業評估，而非以單一保健成分處理。

## 補充品常見形式

市售的納豆激酶多為萃取純化後的膠囊或錠劑，常見以每日 2,000、3,000、3,700 或 4,000 FU 標示活性，也常與紅麴、Q10、魚油等做成「循環保養」類的複方產品。 <a href="#ref-11">[11]</a> 在台灣，本土發酵產業相當成熟：台鹽生技以自有發酵技術與發明專利推出高單位（每顆 3,700 FU）的納豆激酶產品， <a href="#ref-9">[9]</a> 金穎生物科技（GeneFerm）等原料商深耕微生物發酵逾 20 年，具 GMP、FSSC22000 等規範，供應納豆激酶等發酵機能原料並承接 ODM/OEM。 <a href="#ref-10">[10]</a> 這些屬於食品或保健食品，依規不得宣稱療效；選購時可留意 FU 標示、是單方或複方、是否有第三方檢驗，而非被「高單位」數字主導。複方雖看似完整，但同時含有其他影響凝血的成分（如紅麴、魚油）時，更需注意整體的出血風險並與藥師確認。

## 安全性與注意事項

納豆激酶最需要強調的是出血風險。由於它本身以纖溶為主、方向上帶輕度抗凝作用，與抗凝血藥（如 warfarin、heparin）或抗血小板藥（如 aspirin、clopidogrel）併用時，可能疊加出血風險。 <a href="#ref-6">[6]</a> <a href="#ref-7">[7]</a> 已有案例報告指出：有機械瓣膜置換者以納豆激酶取代 warfarin 後發生瓣膜血栓、需再次手術；也有在 aspirin 治療之上加用納豆激酶後發生顱內出血的個案。 <a href="#ref-6">[6]</a> 因此，正在服用抗凝血或抗血小板藥物者，切勿自行併用或以納豆激酶取代處方藥；安排手術或侵入性檢查前，也應主動告知醫師目前使用的保健食品，以評估是否需要停用。新竹臺大分院藥劑部等醫院衛教也提醒，長期服用 warfarin 者若要使用任何保健食品或中藥，務必先告知醫師或藥師，以避免交互作用。 <a href="#ref-8">[8]</a> 此外，對大豆或納豆過敏者不宜使用；孕婦、哺乳者與有出血性疾病史者的安全資料有限，不建議自行使用。任何與疾病或用藥相關的判斷，都應交由醫師與藥師處理。 <a href="#ref-6">[6]</a>

## 常見迷思

- 「納豆激酶能溶解血管斑塊、清血管、通血路」：最嚴謹的長期雙盲試驗（NAPS，Hodis 等 2021，追蹤約 3 年）發現它對亞臨床動脈硬化與血壓與安慰劑沒有顯著差異；被行銷引用來支持「縮小斑塊」的研究（如 Frontiers in Cardiovascular Medicine 2022 一項 1,062 人的臨床研究）為回溯性研究、缺乏對照組，證據等級較低。 <a href="#ref-4">[4]</a> <a href="#ref-5">[5]</a>
- 「可以取代抗凝血藥」：這是最危險的誤解。已有以納豆激酶取代 warfarin 而發生血栓、需再手術的案例；它不能取代醫師處方的抗凝血或抗血小板藥物。 <a href="#ref-6">[6]</a>
- 「FU 越高、效果越強」：FU 是體外分解纖維蛋白的活性單位，並非藥品標準化單位，也不等同臨床療效；不同產品之間不宜單看 FU 數字比較。 <a href="#ref-11">[11]</a>
- 「吃納豆激酶等於吃納豆」：傳統納豆同時含有豐富的維生素 K2，會影響 warfarin 的作用，萃取型產品的 K2 含量通常較低（依產品而異），兩者不能直接畫上等號。 <a href="#ref-8">[8]</a>
- 「機制有纖溶活性，代表能預防心梗、中風」：試管與動物的纖溶活性屬機制層面，人體證據多為小樣本、短期的替代指標，尚無大型長期試驗證明能降低心肌梗塞或中風等事件。 <a href="#ref-4">[4]</a>
