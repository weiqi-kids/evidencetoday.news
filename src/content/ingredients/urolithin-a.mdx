## 30 秒認識

- 尿石素A（Urolithin A，簡稱 UA）是與粒線體有關的代謝物，本身不存在於任何食物裡；它要靠腸道菌把石榴、核桃、莓果中的鞣花單寧（ellagitannins）先水解成鞣花酸，再一步步轉成尿石素A。 <a href="#ref-7">[7]</a>
- 能不能有效轉化取決於個人腸道菌相：在一項健康成人研究中，基線只有約 12% 的人血中測得到尿石素A，喝了石榴汁後也只有約四成能有效轉化。 <a href="#ref-7">[7]</a> 幾乎無法自行生成尿石素A 的代謝型（分類為 UM-0）盛行率因族群與飲食而異，直接補充可繞過這個個體差異。 <a href="#ref-12">[12]</a>
- 目前人體試驗多由業界（Amazentis／Timeline，即 Mitopure 業者）主導，樣本數僅數十人、療程 4 週到 4 個月，且有幾次主要終點其實未達統計顯著。 <a href="#ref-2">[2]</a> <a href="#ref-3">[3]</a>
- 首次人體試驗（Andreux 等 2019 於 Nature Metabolism）主要證明的是安全與生體可用性，不等於臨床療效。 <a href="#ref-1">[1]</a>
- 美國 FDA 給予尿石素A GRAS（一般公認安全）狀態，臨床試驗中耐受性良好；但 GRAS 屬食品安全認定，並非療效核准，也不等同台灣核准。 <a href="#ref-8">[8]</a>

## 基本介紹

尿石素A（Urolithin A）屬於一群稱為尿石素（urolithins）的腸道菌代謝物。石榴、核桃、覆盆子、草莓等富含鞣花單寧的食物進入腸道後，會先被水解成鞣花酸（ellagic acid），再由特定腸道菌代謝成尿石素A。 <a href="#ref-7">[7]</a> 也就是說，餐盤上的石榴或核桃並不直接提供尿石素A，而是提供它的前驅物；能不能真正生成，取決於腸道裡有沒有足夠的轉化菌。 <a href="#ref-7">[7]</a> 2025 年台灣多家生醫媒體（如 GeneOnline）以「粒線體回春／抗老」大量報導它，標題語彙常明顯超前人體證據的等級，這也是本文把「主要終點是否達標」逐一標清楚的原因。 <a href="#ref-10">[10]</a>

## 這是什麼？

標示上多寫作 Urolithin A 或縮寫 UA。要先建立兩個觀念：第一，它是腸道菌的「代謝產物」，屬於可選的補充成分，並非人體必需營養素，沒有每日必需建議量；第二，它有明顯的「代謝型」個體差異——依腸道菌相不同，一般把人分成能大量生成尿石素A（UM-A）、生成其他尿石素（UM-B），以及幾乎不生成（UM-0）三型。 <a href="#ref-12">[12]</a> 其中 UM-0 這型幾乎不生成尿石素A，盛行率因族群、年齡與飲食而有明顯差異，光靠多吃石榴也很難補起來。這種個體差異，正是業界推出「直接補充尿石素A」產品的立論基礎。

## 在身體裡做什麼？／主要作用機制

尿石素A 最常被討論的機制是「粒線體自噬（mitophagy）」——協助細胞清除受損的粒線體、維持粒線體品質。在臨床前研究中，Ryu 等 2016 於 Nature Medicine 發現它能在線蟲誘導粒線體自噬、延長壽命，並改善囓齒類的肌肉與運動功能。 <a href="#ref-11">[11]</a> 回到人體，Andreux 等 2019 於 Nature Metabolism 招募 60 名健康久坐長者，以單劑 250 至 2000 毫克遞增、多劑每日 500 與 1000 毫克連續 4 週給予；4 週後 500 與 1000 毫克可調節血漿醯基肉鹼與骨骼肌粒線體相關基因表現，但這些都屬次要終點，該研究主要確立的是安全與生體可用性。 <a href="#ref-1">[1]</a> 2024 年 Ageing Research Reviews 的系統性回顧也指出，目前人體效益多停留在發炎與粒線體基因等替代終點。 <a href="#ref-6">[6]</a> 機制與生物標記的變化，並不等於跌倒、失能等硬臨床結果的改善，兩者之間仍需更大型試驗銜接。

## 食物來源／常見來源

尿石素A 本身不存在於食物中，真正能從飲食取得的是它的前驅物——鞣花單寧與鞣花酸，主要來自石榴、核桃、覆盆子、草莓、黑莓與開心果等。 <a href="#ref-7">[7]</a> 由於最終能否生成尿石素A 取決於個人腸道菌相，「多吃石榴＝一定補得到尿石素A」的推論並不成立：同樣一杯石榴汁，UM-A 型的人血中尿石素A 明顯上升，UM-0 型的人幾乎沒有變化。 <a href="#ref-7">[7]</a> 從飲食角度，石榴、莓果、堅果的價值更多來自整體營養與多酚，而非單一鎖定尿石素A。

## 建議攝取量／常見攝取建議／研究中常見使用方式

尿石素A 沒有每日必需攝取量。人體試驗常見的給法是：單劑 250 至 2000 毫克用於安全與藥動評估，多劑則以每日 500 或 1000 毫克、連續 4 週到 4 個月最常見。 <a href="#ref-1">[1]</a> 針對肌肉功能的試驗多採每日 500 或 1000 毫克。 <a href="#ref-5">[5]</a> 提醒兩點：一是劑量更高不代表效果更好，多數正面變化仍停留在生物標記層級；二是這些劑量對應的是研究情境，個人是否需要補充、要用多少，宜交由醫師或營養師依狀況評估。

## 缺乏風險或攝取不足問題

尿石素A 不是必需營養素，沒有「缺乏症」——不補充它並不會造成任何已知的健康缺損。前面提到的 UM-0 代謝型，指的是「無法自行由食物轉化生成」，這是菌相差異，並不代表身體缺了某種營養素需要矯正。把肌力下降、疲倦或老化直接歸因於「尿石素A 不足」並不合理，這些更受年齡、運動量、睡眠與整體飲食影響。想嘗試補充的人，較合理的定位是「可選的、以粒線體與肌肉為訴求的補充品」，並清楚它目前的人體證據仍屬早期。

## 補充品常見形式

市面上尿石素A 多以膠囊或粉包流通，原料常見石榴萃取，或標示授權自 Mitopure（Amazentis／Timeline）的合成尿石素A。 <a href="#ref-2">[2]</a> 有些產品把它與膠原蛋白、維生素 D、Omega-3 等做成抗老或運動複方；複方看似完整，卻讓判讀變難——用了之後若有變化不易歸因，若有不適也較難找出來源。選購時可留意每份的尿石素A 毫克數（多對應研究採用的 500 或 1000 毫克）、原料來源，以及是否有第三方檢驗，而非被「回春」「神藥」等文案帶著走。

## 安全性與注意事項

就目前資料，尿石素A 的耐受性大致良好：多項隨機對照試驗在每日 1000 毫克、最長 4 個月下，與安慰劑相比未出現顯著的不良事件差異，最常見的是輕微腸胃不適。 <a href="#ref-3">[3]</a> 2025 年 Nature Aging 的 4 週試驗中，受試者的肝、腎功能指標也無顯著變化。 <a href="#ref-4">[4]</a> 美國 FDA 已把尿石素A 列為 GRAS，允許用於優格、蛋白飲等，每份最高 500 至 1000 毫克。 <a href="#ref-8">[8]</a> 但要強調：GRAS 是「食品安全」認定，並非「療效」核准，也不等同台灣核准。依台灣《健康食品管理法》，未取得許可證（俗稱小綠人）不得宣稱保健功效，任何產品一律不得宣稱治療或矯正疾病的醫療效能。 <a href="#ref-9">[9]</a> 孕婦、哺乳者、慢性病或長期用藥者、準備手術者的專門安全資料仍有限，開始補充前宜先諮詢醫師或營養師。

## 常見迷思

- 「吃石榴、核桃就能補到尿石素A」：食物提供的是前驅物，能否生成取決於腸道菌相；在一項健康成人研究中，喝石榴汁後也只有約四成的人能有效轉化，且完全無法自行生成的 UM-0 代謝型比例因族群而異。 <a href="#ref-7">[7]</a> <a href="#ref-12">[12]</a>
- 「臨床證明肌力提升 12%、能抗老回春」：Singh 等 2022 於 Cell Reports Medicine 的中年成人試驗，主要終點峰值功率相對安慰劑未達顯著；次要終點握力（1000 毫克組自基線約 +5.1%，p=0.08）也未達顯著。常被引用的「約 +12%」其實是「大腿後側（膕旁肌）肌力」這個次要終點，它在兩個劑量下相對安慰劑確實達統計顯著（500 毫克 p=0.027、1000 毫克 p=0.029），但這是次要而非主要終點，也不等於抗老回春。另在 Liu 等 2022 於 JAMA Network Open 的高齡試驗，六分鐘步行距離則未達顯著。 <a href="#ref-2">[2]</a> <a href="#ref-3">[3]</a>
- 「FDA 認證有效」：FDA 給的是 GRAS 食品安全認定，不涉及療效核准，更不等同台灣核准。 <a href="#ref-8">[8]</a>
- 「粒線體自噬＝人會變年輕」：粒線體自噬是細胞層級的機制，2025 年 medRxiv 的系統性回顧（僅 3 篇試驗、174 人）結論為現階段證據不足以支持用尿石素A 改善肌肉功能，離「抗老回春」還很遠。 <a href="#ref-5">[5]</a>
