# Agent Instructions

本專案為健康議題編輯平台「本日有據」（evidencetoday.news）。

## 套件管理器

使用 **pnpm**（不是 npm）。

## 常用任務

### 撰寫趨勢文章

當收到「撰寫趨勢文章」指令時，依 README.md 的「AI 任務：撰寫趨勢文章」章節執行完整 7 步驟流程：

1. 準備（讀取 config + 去重紀錄）
2. 搜尋素材（8 組查詢）
3. 評分與選題（五維度加權）
4. 撰寫文章（完整 frontmatter + 口語化內文）
5. **自動審核**（build 驗證 + frontmatter 完整性 + 內容品質，不通過就修正，反覆直到全部通過，最多 5 輪）
6. 更新去重紀錄
7. 發布（push main 或開 PR）

**關鍵要求：步驟 5 的審核必須全部通過才能進入步驟 6。** 開出的 PR 應該是可以直接 merge 的狀態。

### 重要規則

- 日期一律使用台灣時間（UTC+8）
- tags 禁止含 `/`（會導致 build 失敗）
- 禁止聳動用語、具體醫療建議、醫療承諾
- 使用台灣繁體中文，禁止中國用語
- 修改程式碼前必須先讀取完整檔案
- Schema 定義在 `src/content.config.ts`
- 完整 SOP 見 `docs/news_sop.md`
