# Local Preview

1. Install dependencies:
   - `pnpm install`
2. Start dev server:
   - `pnpm dev`
3. Open browser:
   - `http://localhost:4321`

Key pages:
- Home: `/`
- Search: `/search/`
- Editor: `/authors/luo-yang/`
- Disclosure: `/disclosure/`
- Podcasts: `/podcasts/`
- News: `/news/`

Build preview:
1. `pnpm run build`
2. `pnpm exec astro preview`
3. Open `http://localhost:4321`
