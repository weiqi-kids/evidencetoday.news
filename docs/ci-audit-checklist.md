# 本日有據 CI 視覺一致性檢查清單

## 1. 全站共通

- [ ] 是否仍像健康媒體，而不是商城、診所或產品頁？
- [ ] 是否使用既有 tokens？
- [ ] 是否沒有新增硬寫 hex？
- [ ] 是否沒有使用外部 CDN 字體？
- [ ] 是否沒有用恐懼或銷售語氣包裝健康內容？

## 2. 色彩

- [ ] 主背景是否以 paper / white / neutral 為主？
- [ ] teal / navy 是否用於品牌識別與結構重點？
- [ ] coral 是否只作為少量 CTA 或提醒？
- [ ] 分類色是否只用於 badge、頂線或小面積識別？

## 3. 字體

- [ ] 是否未更動既有字體設定？
- [ ] 標題、正文、UI 文字是否符合目前使用邏輯？
- [ ] 是否沒有新增外部字體？

## 4. 卡片

- [ ] 卡片是否使用既有 radius / border / shadow？
- [ ] 卡片是否像媒體內容入口，而不是商品卡？
- [ ] 不同卡片之間的間距、標題、metadata 是否一致？

## 5. Logo

- [ ] Logo 是否未被拉伸、變色或加不必要陰影？
- [ ] TopNav 是否維持 icon + 中文名稱？
- [ ] Footer 是否維持合理品牌識別？

## 6. 圖像

- [ ] 圖像是否避免商品瓶罐銷售感？
- [ ] 圖像是否避免醫療恐嚇？
- [ ] 圖像是否避免十字架元素，除非使用者明確要求？
- [ ] 圖像是否符合 editorial / evidence / clean / calm 方向？

## 7. OG 圖

- [ ] OG 圖是否有分類 badge？
- [ ] 標題是否足夠大？
- [ ] 是否保留本日有據品牌識別？
- [ ] 是否沒有商品銷售感？
- [ ] LINE / Facebook 預覽是否不被裁切？

## 8. 手機版

- [ ] 375px 是否無水平 overflow？
- [ ] 標題是否不擠？
- [ ] 卡片與按鈕是否有足夠間距？
- [ ] 圖片與影片是否不破版？

## 9. 頁面類型

- [ ] 首頁是否仍有媒體入口感？
- [ ] 文章頁是否重點清楚、引用來源可查？
- [ ] 闢謠頁是否判讀清楚但不情緒化？
- [ ] 成分解析頁是否像知識庫，不像商品頁？
- [ ] Podcast 頁是否像節目內容頁，不只是播放器清單？
- [ ] 短影音頁是否能逐步導向站內內容資產？
- [ ] 搜尋頁是否保持簡潔、可用？

## 10. 不應出現的視覺方向

- [ ] 不像保健食品銷售頁
- [ ] 不像診所廣告頁
- [ ] 不像政府宣導頁
- [ ] 不像電商分類頁
- [ ] 不像 AI 生成模板站
- [ ] 不像個人履歷頁
